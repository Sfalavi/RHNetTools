makeRegionVar {RHNetTools}R Documentation

Recodes a vector of House IDs into region IDs.

Description

Region 1: NC
Region 2: TX
Region 3: OR

Usage

makeRegionVar(pHIDs)

Arguments

pHIDs

A vector of House ID #'s (integers) to be assigned to regions

Value

A vector of region numbers the same length as the input vector
