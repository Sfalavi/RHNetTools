replaceNaNWithNA {RHNetTools}R Documentation

Substitutes NA for NaN in a data.table

Description

If input is a data.frame, it will be converted to a data.table (which is also a data frame)

Usage

replaceNaNWithNA(pInTB)

Arguments

pInTB

Any tibble or data.frame

Value

Returns the input tibble or df with NaNs replaced by NAs
