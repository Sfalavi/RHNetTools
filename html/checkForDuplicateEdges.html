checkForDuplicateEdges {RHNetTools}R Documentation

Checks long-format data (output from 'makeLongNet') for duplicate edges

Description

Checks long-format data (output from 'makeLongNet') for duplicate edges

Usage

checkForDuplicateEdges(rlpNet, pWave = 0)

Arguments

rlpNet

(tbl_df) Long-format edge list of the form 'longNet' or 'rlpNet.

pWave

(scaler, default=0) The wave to select. If default, function assumes rlpNet is for 1 wave only, & that is the wave to use. Otherwise the function will throw an error.

Value

Outputs a tbl_df showing duplicated rows, in the same format as 'rlpNet'
