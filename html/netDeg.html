netDeg {RHNetTools}R Documentation

Calculate average in + out ("Freeman") degrees across a whole network

Description

Calculate average in + out ("Freeman") degrees across a whole network

Usage

netDeg(pNet)

Arguments

pNet

('network') A 'network' class object (from pkg 'network'). Could be binary or non-binary.

Details

Wraps the sna function 'degree', which gives a vector of the total in + out-degree values by node.

Value

(numeric) A numeric variable that averages.
