Description

Converts a relationship "short name" string (e.g. FR, LO, etc. ) to a full (and more descriptive) string (e.g. friend, loan, etc. )

Usage

assignRlpName(pRlpShortName)

Arguments

pRlpShortName

The two-character 'short name' code for each relationship type

Value

The name as a string
