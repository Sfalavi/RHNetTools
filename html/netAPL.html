netAPL {RHNetTools}R Documentation

Calculate average path (geodesic) length for a network

Description

Calculate average path (geodesic) length for a network

Usage

netAPL(pNet)

Arguments

pNet

('network') A 'network' class object (from pkg 'network'). Could be binary or non-binary.

Details

Wraps the sna function 'geodist', which gives a matrix of the pairwise distances, not the average. This function only includes connected pairs.

Value

(numeric) A numeric variable that averages.
